# 📐 Cuaderno Virtual — Dibujo Técnico

> Espacio digital para organizar y compartir el material académico de la especialidad de **Dibujo Técnico**: apuntes, investigaciones, planchas, ejercicios y exámenes.

---

## 🔗 Ver sitio

🌐 [cuaderno-virtual-dt](https://wilmerbotina132.github.io/cuaderno-virtual-dt)

---

## 📋 Contenido

| Sección | Descripción |
|---|---|
| 📝 **Apuntes** | Notas y resúmenes de clases |
| 🔬 **Investigaciones** | Aportes e investigaciones por tema |
| 📏 **Planchas y Planos** | Trabajos de representación técnica |
| ✏️ **Ejercicios** | Prácticas y actividades resueltas |
| 📄 **Exámenes** | Material de evaluaciones |

## 📁 Estructura del proyecto

```
├── public/              # Archivos estáticos (imágenes, íconos)
├── src/
│   ├── app/             # Rutas con App Router
│   │   ├── about/       # Presentación
│   │   ├── blog/
│   │   │   └── [slug]/  # Entradas individuales (apuntes, planchas, etc.)
│   │   ├── contact/     # Contacto
│   │   ├── layout.tsx
│   │   └── page.tsx     # Inicio
│   ├── components/      # Componentes reutilizables
│   ├── data/            # Contenido del cuaderno
│   ├── hooks/           # Custom hooks
│   └── styles/          # Estilos globales
```

## 🛠️ Instalación local

```bash
# Clonar el repositorio
git clone https://github.com/WilmerBotina132/cuaderno-virtual-dt.git
cd cuaderno-virtual-dt

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.
